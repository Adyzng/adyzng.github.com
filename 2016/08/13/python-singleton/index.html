Python设计模式-单例模式

发表于 2016-08-13   |   分类于 Python ， 设计模式   |     |   阅读量 次

定义

单例（singleton）模式标准定义：保证一个类仅有一个实例，并提供一个访问实例对象的全局方法。
属于创建行模式，采取一定的方法保证在整个系统中，单例类只有一个实例可用。

应用

单例模式应用的场景一般发现在以下条件下：

  • 资源共享的情况下，避免由于资源操作时导致的性能或损耗等。如日志文件，应用配置。
  • 控制资源的情况下，方便资源之间的互相通信。如线程池等。

例如：

  • Windows的Task Manager（任务管理器）就是很典型的单例模式，你只能打开一个Windows Task Manager；
  • 网站计数器，一般也是采用单例模式实现，否则难以同步；
  • 日志应用，一般都使用单例模式实现，由于共享的日志文件一直处于打开状态，因为只能有一个实例去操作；
  • Web应用的配置对象，一般也应用单例模式，这个是由于配置文件是共享的资源；
  • 多线程的线程池的设计一般也是采用单例模式，这是由于线程池要方便对池中的线程进行控制；
